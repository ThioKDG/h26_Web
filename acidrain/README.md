# ☔ 산성비 타자연습

위에서 떨어지는 단어를 바닥에 닿기 전에 빠르게 입력해서 제거하는 한글 타자 연습 게임입니다.

🎮 **[지금 플레이하러 가기](https://thiokdg.github.io/h26_Web/acidrain/)**

---

## 🎯 게임 방법

빗방울처럼 떨어지는 단어가 화면 하단의 빨간 **데드라인**에 닿기 전에 정확히 타이핑해서 제거하세요.

- 페이지 입력창에 **그냥 타이핑을 시작하면** 게임이 시작됩니다 (별도 시작 버튼 없음)
- 단어를 정확히 입력하면 단어가 사라지고 점수 획득
- 단어가 데드라인에 닿으면 **HP -1** — HP가 0이 되면 게임오버
- HP는 최대 5, 화면 좌상단에서 확인 가능

### 점수와 콤보

```
점수 = 단어 길이 × 콤보 배수(최대 5) × 10
```

연속으로 정답을 맞추면 콤보가 쌓이고, 한 번 놓치면 콤보는 0으로 리셋됩니다. **800점마다 레벨업**하면서 단어 풀이 어려워지고 떨어지는 속도가 빨라집니다.

### 게임 통계

게임이 끝나면 화면에 다음 정보가 표시됩니다:

- **분당 타수** — 한글 자모 단위로 정확히 카운트 (받침 있으면 3타, 없으면 2타, 영문은 1타)
- **정확도** — 맞춘 단어 / 전체 시도 단어
- **평균 반응 시간** — 단어가 등장한 시점부터 매칭 완료까지의 평균 (ms)

한컴타자연습이나 한글 타자 자격 시험과 동일한 방식으로 측정되므로, 본인의 실력을 객관적으로 확인할 수 있습니다.

## 🌈 크레이지 모드

우측 상단 토글로 켜고 끌 수 있는 하드 모드입니다.

- 4글자 이상의 어려운 단어만 등장
- 낙하 속도가 **수시로 빨라졌다 느려졌다** 합니다 (사인파 진동)
- HUD와 입력바가 무지개색으로 빛남
- 일반 모드와 **랭킹이 따로 저장**됨

### ⚠ 함부로 끄지 마세요

게임 도중에 크레이지 모드를 끄려고 클릭하면 **시야 차단(BLIND) 패널티**가 발동됩니다.

- 클릭당 +2초 누적, 최대 8초
- 화면이 거의 새까맣게 변해 단어가 안 보임
- 점수와 HP는 계속 그대로 — 안 보이는 단어를 감으로 입력해야 함
- HUD는 그대로 보이므로 패널티 잔여 시간은 확인 가능

## 💎 파워업 단어

게임 중 가끔(5% 확률, 레벨 2 이상부터) 빛나는 특수 단어가 등장합니다.

| 단어 | 효과 |
|------|------|
| **❄ 시간정지** | 3초간 모든 단어와 새 단어 생성이 멈춤 |
| **💣 폭탄제거** | 화면에 있는 모든 일반 단어 제거 + 글자수 보너스 점수 |
| **💖 체력회복** | HP +1 (이미 최대치면 +200 점수 보너스) |

파워업 단어는 일반 단어보다 약간 느리게 떨어지고, 놓쳐도 HP가 깎이지 않습니다. 한 글자도 빠짐없이 입력해야 발동됩니다 (예: 시간정지 효과를 받으려면 `시간정지` 입력).

---

## 🏆 랭킹 시스템

게임오버 시 점수가 TOP 10 안에 들면 신기록 화면에서 이름을 등록할 수 있습니다.

- TOP 10 리스트는 **시작 화면**과 **게임오버 화면**에서 확인 가능
- **일반 모드와 크레이지 모드 랭킹이 분리**되어 있어 공정한 경쟁
- 데이터는 **브라우저 로컬에만** 저장됨 (다른 사람과 공유되지 않음)
- 본인의 최신 등수는 노란 하이라이트로 표시

## 📝 내 단어 추가하기

우측 상단 **📝 내 단어 관리** 버튼을 누르면 본인이 원하는 단어를 자유롭게 추가할 수 있습니다.

- 난이도별(🟢 쉬움 / 🟡 보통 / 🔴 어려움)로 분류
- 추가하면 즉시 게임에 반영
- 친구 이름, 좋아하는 단어, 코딩 용어 등 무엇이든 OK
- 잘못 추가했으면 ✕ 버튼으로 삭제

추가된 단어는 기본 단어 풀에 섞여서 등장하며, 4글자 이상은 자동으로 크레이지 모드 풀에도 포함됩니다.

## ⌨ 조작법

| 키 / 버튼 | 기능 |
|----------|------|
| 일반 타자 | 단어 입력 (한글 IME 지원) |
| `ESC` | 일시정지 ↔ 재개 토글 |
| `Enter` | 신기록 시 이름 등록 |
| `다시 시작` 버튼 | 게임오버 후 새 게임 시작 |
| 🌈 우측 상단 | 크레이지 모드 토글 |
| 📝 우측 상단 | 단어 관리 모달 |

첫 방문 시 5장짜리 **튜토리얼 모달**이 자동으로 뜹니다. 한 번 보면 다시 표시되지 않으며, 키보드 `←` `→`로 슬라이드 이동, `Enter`로 진행, `ESC`로 닫을 수 있습니다.

## 💡 플레이 팁

- **콤보를 끊지 마세요** — 5콤보부터 점수가 5배. 한 단어 놓치면 0으로 리셋됩니다.
- **데드라인이 빨갛게 빛날 때 침착하게** — 빨간 구간에 들어간 단어는 시간이 부족하지만, 오타를 내면 더 손해입니다. 차분히 정확히 치세요.
- **파워업에 욕심 부리지 마세요** — 일반 단어를 처리하다가 가시거리 안에 들어오면 잡는 정도가 좋습니다. 파워업 잡으려다 다른 단어 놓치면 손해.
- **크레이지 진동 패턴을 활용하세요** — 속도가 잠시 느려지는 순간을 노려 긴 단어를 입력하면 이득.
- **자기 단어를 등록하면 콤보가 늘어납니다** — 익숙한 단어가 많이 등장할수록 반응 속도가 빨라집니다.
- **시야 차단 중에도 패닉 금지** — 8초만 버티면 풀립니다. HP와 입력창은 보이니 이전 단어를 외워서 치는 연습이 됩니다.

## 🛠 기술

빌드 도구 없이 순수 정적 웹으로 만들어졌습니다.

- **HTML5 + CSS3** (커스텀 프로퍼티, 키프레임 애니메이션)
- **HTML5 Canvas** (단어, 배경, 빗방울 렌더링)
- **Vanilla JavaScript** (ES6+ 모듈, 클래스)
- **localStorage** (랭킹, 사용자 단어, 설정, 튜토리얼 진행 상태 영속화)

서버나 데이터베이스 없이 동작하기 때문에 GitHub Pages 같은 정적 호스팅에 그대로 올릴 수 있습니다.
